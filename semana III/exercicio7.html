Regexp

Crie regex de validação para cada campo abaixo. A validação deve ser executada no evento blur de cada campo. crie seu código o mais genérico possível, a fim de evitar repetições. Após validar o campo, caso haja erro, adicionar ao campo a classe de estilo "error", que fará com que o campo fique com borda vermelha. Caso não haja erro, remover a classe "error".

As validações adicionadas devem ser:

  • Nome: apenas letras e deve começar com letras maiúcula.
  • Sobrenome: a mesma validação de nome
  • Idade: apenas números positivos (sem sinal)
  • Email: formato aaaa@bbbb.ccc, onde aaaa pode conter letras, . e _, bbb apenas letras e ccc apenas letras também.
  • CPF: DDD.DDD.DDD-DD, onde D é qualquer dígito

BONUS: Se puder, após conseguir criar as regexp, faça algo orientado a objetos, como se pudesse ser utilizado em qualquer projeto. Como se fosse utilizar da seguinte forma, e a validação já seria configurada no evento "blur" dos inputs: 

      Validations.configure ( "email", Validations.EMAIL );
      Validations.configure ( "idade", Validations.POSITIVE_INTEGER );
